# Aviso de conteúdo SRD

O Atlas das Cinzas não distribui criaturas proprietárias que estejam fora do System Reference Document.

A sincronização opcional utiliza a D&D 5e SRD API configurada em `SRD_API_BASE_URL` para obter conteúdo do SRD das regras de 2014.

Dungeons & Dragons SRD 5.1 é disponibilizado sob Creative Commons Attribution 4.0 International.

O mestre pode cadastrar ou importar conteúdo próprio, homebrew ou material para o qual possua autorização. O projeto não inclui catálogos proprietários externos.
